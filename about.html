About CINEMORA

CINEMORA is a student-built movie review platform that blends cinematic passion with full-stack creativity. Whether you're into mass Tamil action or soulful Hollywood drama, CINEMORA lets you explore, rate, and review your favorite films with style.

This project was crafted by DivyaKumar and Team, a BTech IT student with a flair for remixing entertainment content and building scalable web apps. From frontend finesse to backend logic, every line of code reflects personal branding and a love for cinema.

What powers CINEMORA:

  • HTML, CSS, JavaScript for the frontend
  • Node.js + Express + MongoDB for backend (coming soon)
  • Free hosting solutions for deployment
  • Custom search logic and review system
